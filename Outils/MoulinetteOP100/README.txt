MoulinetteOP100

Permet de mouliner les fichiers .txt à l'origine des PV OP100 manuel.
Attention cependant, cette version n'est pas compatible avec tous les formats de fichiers .txt
Si ajout de test ou autre, ce Vi perdra en précision (voire donnera des données inexploitable).

Pour utilisation :

Placer dans le dossier "Textes" l'ensemble des fichiers à mouliner.

Dans le Vi référencer le chemin vers ce dossier puis RUN.

La table se rempli. Vous pouvez faire un clique droit "Exporter vers Excel" sur la table pour traiter les données.
